Project 07

TechNova — Store Website

HTML CSS JavaScript PHP MySQL HMI / Ergonomics UX Design

Overview

A fully functional e-commerce website for an electronics store called TechNova, built with a PHP/MySQL backend and a carefully designed frontend. The project's focus went beyond basic functionality — it was an applied study of HMI (Human-Machine Interface) ergonomics: color psychology, visual hierarchy, user guidance, and form clarity. Every design decision — from the dark gradient palette to the step-by-step checkout instructions — was made to reduce cognitive load and guide the user naturally through the experience.

Color Palette — HMI Design Choice

The palette was chosen deliberately to reflect a premium technology brand. The dark navy-to-crimson gradient creates a sense of depth and sophistication, while the warm red accent draws attention to key actions (Login/Register button). These are not arbitrary choices — they follow HMI principles of contrast, attention hierarchy, and emotional tone.

HMI Ergonomics Applied

🎨
Color Psychology
Dark background reduces eye strain during browsing sessions. Red CTA buttons trigger urgency and draw immediate attention to key actions.
🧭
User Guidance
Forms include clear labels and inline instructions — e.g., "Enter your full name", "Date of birth (DD/MM/YYYY)" — eliminating ambiguity at every input step.
📐
Visual Hierarchy
Categories are presented as large, equally-spaced cards with images and brief descriptions — guiding the eye naturally without overwhelming the user.
✅
Feedback & Confirmation
A dedicated confirmation page for purchases provides clear order summary and success feedback — preventing user anxiety after completing a transaction.
⚡
Consistency
Navigation, button styles, and color usage are consistent across all pages — reducing the cognitive load needed to learn the interface.
🔍
Affordance
Interactive elements (buttons, category cards) are visually distinct from static content — making it immediately clear what is clickable.

Key Features

  • Full user authentication — register, login, session management in PHP
  • Product catalog organized by category with MySQL backend
  • Shopping cart and purchase confirmation flow
  • HMI-focused forms — clear field labels, inline guidance, date format hints
  • Consistent dark gradient design system across all pages
  • Responsive layout adapting to different screen sizes

What I Learned

This project was my first deep dive into HMI ergonomics as a discipline — not just building something that works, but building something that feels right to use. Choosing the color palette, writing clear form instructions, and structuring the visual hierarchy taught me that good UI is about reducing friction at every step. Combining that with a full PHP/MySQL backend gave me end-to-end experience in building a real web product.

Project 05

DSS Website

HTML CSS JavaScript PHP MySQL Decision Support System

Overview

A web-based Decision Support System (DSS) for automated recruitment, built with a full PHP/MySQL stack. The system allows administrators to manage candidates and evaluate their job applications using a weighted scoring model that considers academic degree, professional experience, and skills. The final score is compared against a configurable threshold to automatically accept or reject each candidate — with a downloadable PDF report generated for each evaluation.

Decision Algorithm — Weighted Scoring Model

Final Score Formula
Final Score = (Degree Score × W₁) + (Experience Score × W₂) + (Skills Score × W₃)
W₁ + W₂ + W₃ = 1.0 (100%)
W₁ Degree = 20% W₂ Experience = 30% W₃ Skills = 50% Pass Threshold = 60 / 100

Each component is normalized to a 0–100 range before weighting. Degree is a fixed point score (PhD = 50, Master = 30, Bachelor = 20). Experience is normalized against a 120-month maximum. Skills is normalized against a maximum of 5 skills. Admins can adjust all weights and the passing threshold per job role. Candidates scoring ≥ 60 are automatically Accepted; below that, Rejected.

Key Features

  • Admin authentication with role-based access to the dashboard
  • Candidate registration with degree, experience, and multi-skill selection
  • Configurable weights — admin can tune W₁/W₂/W₃ per job requirement
  • Automated scoring with instant Accept / Reject decision
  • Detailed score breakdown: degree, experience, and skills components
  • Downloadable PDF report generated per evaluation
  • Many-to-many candidate ↔ skills relationship (junction table)
  • 12 pre-loaded skills across Programming, Web, Database, and DevOps

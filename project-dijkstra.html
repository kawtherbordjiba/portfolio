Project 08

Dijkstra Algorithm Program

C Graph Algorithms Shortest Path Data Structures Greedy Algorithm

Overview

Implementation of Dijkstra's shortest path algorithm in C, applied to a weighted directed graph represented as an adjacency matrix. The program computes the minimum distance from a chosen source vertex to all other vertices in the graph, and reconstructs the full shortest path using a predecessor array. This is a fundamental greedy algorithm widely used in network routing protocols such as OSPF and IS-IS.

Language
C
Type
Greedy Algorithm
Graph
Weighted, Directed
Representation
Adjacency Matrix
Input
Terminal / stdin

How the Algorithm Works

1
Initialize distances
Set distance to source = 0, all others = ∞. Mark all vertices unvisited.
2
Pick the unvisited vertex with minimum distance
This is the greedy step — always extend the shortest known path first.
3
Relax all neighboring edges
For each neighbor, if dist[u] + weight(u,v) < dist[v], update dist[v] and record predecessor.
4
Mark current vertex as visited
A visited vertex's shortest distance is finalized — it will never be updated again.
5
Repeat until all vertices are visited
Backtrack through the predecessor array to reconstruct and print the full shortest path.

Key Features

  • Adjacency matrix input — reads weighted graph from terminal
  • Predecessor array to reconstruct the full path, not just the distance
  • Handles disconnected nodes — detects unreachable vertices
  • Works on directed and undirected weighted graphs
  • Only works with non-negative edge weights (by design)

Core Implementation

#include <stdio.h>
#define INFINITY 9999
#define MAX      10

void dijkstra(int G[MAX][MAX], int n, int startnode) {
    int cost[MAX][MAX], distance[MAX], pred[MAX];
    int visited[MAX], count, mindistance, nextnode, i, j;

    /* Build cost matrix — replace 0 with INFINITY for non-edges */
    for (i = 0; i < n; i++)
        for (j = 0; j < n; j++)
            cost[i][j] = (G[i][j] == 0) ? INFINITY : G[i][j];

    /* Initialize distance[], pred[] and visited[] */
    for (i = 0; i < n; i++) {
        distance[i] = cost[startnode][i];
        pred[i]     = startnode;
        visited[i]  = 0;
    }
    distance[startnode] = 0;
    visited[startnode]  = 1;
    count = 1;

    while (count < n - 1) {
        /* Greedy step: pick unvisited vertex with min distance */
        mindistance = INFINITY;
        for (i = 0; i < n; i++)
            if (distance[i] < mindistance && !visited[i]) {
                mindistance = distance[i];
                nextnode    = i;
            }

        visited[nextnode] = 1;

        /* Relax edges from nextnode */
        for (i = 0; i < n; i++)
            if (!visited[i] && mindistance + cost[nextnode][i] < distance[i]) {
                distance[i] = mindistance + cost[nextnode][i];
                pred[i]     = nextnode;
            }
        count++;
    }

    /* Print shortest paths and distances */
    for (i = 0; i < n; i++) {
        if (i != startnode) {
            printf("Distance to %d: %d\n", i, distance[i]);
            printf("Path: %d", i);
            j = i;
            while (j != startnode) { printf(" <- %d", pred[j]); j = pred[j]; }
            printf("\n");
        }
    }
}

int main() {
    int G[MAX][MAX], n, u;
    printf("Enter number of vertices: "); scanf("%d", &n);
    printf("Enter adjacency matrix:\n");
    for (int i = 0; i < n; i++)
        for (int j = 0; j < n; j++) scanf("%d", &G[i][j]);
    printf("Enter starting node: "); scanf("%d", &u);
    dijkstra(G, n, u);
    return 0;
}
